# 25HRS 補助星盤 · Grant Astrolabe

政府補助申請時序對照表（互動版）。轉動星盤、滑動時間軸，對齊每個月的補助提案窗口，查看提案時間、截止時間與強化重點。

- 品牌：第廿五時 25HRS 微型創投家加盟計畫
- 補助範圍：競爭型（SBIR、SIIR、SITI）、資格型（WLB 工作與生活平衡）、政策性貸款
- 技術：單一 HTML 檔，無相依套件，桌機與手機 RWD

> 本對照表為教育與規劃參考，實際梯次與截止日以各主管機關當年度官方公告為準。25HRS 協助提升提案品質與通過率，不協助不實申報。
